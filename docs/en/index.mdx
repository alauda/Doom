# Introduction

Doom is a documentation development tool designed for internal use at Alauda, built on top of [rspress](https://rspress.dev/zh/). It provides users with a rich set of built-in plugins for an out-of-the-box experience.

## Core Capabilities

- Automatically generates a configurable weight (order) left sidebar
- Full-text static document search
- Multilingual support

## Based on Markdown and its extension MDX

MDX is a powerful content development approach. You can write Markdown files as you normally would while also using React components within the Markdown content:

![](https://lf3-static.bytednsdoc.com/obj/eden-cn/uhbfnupenuhf/rspress/mdx-intro.png)

For more details, you can refer to the [“Using MDX” documentation](https://rspress.dev/zh/guide/basic/use-mdx).
