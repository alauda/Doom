# Installation

<Overview />

This document provides a detailed description of the installation process for the platform. The platform adopts a **management cluster** + **business cluster** layered architecture design. The management cluster serves as the control hub of the platform, facilitating cluster and tenant management, while the business cluster is responsible for carrying application loads and supports horizontal scaling.
