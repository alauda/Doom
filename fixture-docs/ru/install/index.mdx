# Установка

<Overview />

Этот документ предоставляет подробное описание процесса установки платформы. Платформа использует архитектурное проектирование в слоистом формате **управляющего кластера** + **бизнес-кластера**. Управляющий кластер служит контрольным центром платформы, обеспечивая управление кластерами и арендаторами, в то время как бизнес-кластер отвечает за обработку рабочих нагрузок приложений и поддерживает горизонтальное масштабирование.
